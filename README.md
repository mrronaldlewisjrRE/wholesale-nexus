# WholesaleNexus

A premium real estate wholesaling platform built with React and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js installed ([Download here](https://nodejs.org/))
- Git installed ([Download here](https://git-scm.com/))
- **[See Installation Guide](./INSTALL_GUIDE.md)** for detailed Windows setup.

### Installation

1.  Install dependencies:
    ```bash
    npm install
    npm install lucide-react
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

## Features

- **Dashboard**: Overview of deals, leads, and profit.
- **Zillow Intelligence**: Market search and scraping simulator.
- **Document Center**: Contract templates and e-signature simulation.
- **ROI Analysis**: MAO Calculator based on 70% rule.
- **Legal Hub**: Compliance search and glossary.
- **Pipeline Tracking**: Deal management table.

## Tech Stack

- React
- Tailwind CSS
- Lucide React (Icons)
